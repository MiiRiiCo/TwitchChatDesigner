 - To use this tool properly, please use a Chromium based browser (New Edge / Chrome / New Opera). This tool is optimized for these browsers only. This is because the browser-source in OBS is also Chromium-based.

 - Browsers like Firefox or Safari are NOT supported.

 - If the tool shouldn't work well, try updating your browser. Otherwise try a different browser.

 - If the chat in OBS doesn't work or doesn't show proberly, try updating OBS.
